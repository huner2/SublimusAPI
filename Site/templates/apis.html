API List



User APIs


getUsernameById

Send a GET request to http://SublimusAPI.us.to/apis/getUsernameById/USERID to get username of a player by ID

getIdByUsername

Send a GET request to http://SublimusAPI.us.to/apis/getIdByUsername/USERNAME to get ID of a player by username

userCanManageAsset

Send a GET request to http://SublimusAPI.us.to/apis/userCanManageAsset/USERID/ASSETID to get whether a user can manage an asset or not

userHasAsset

Send a GET request to http://SublimusAPI.us.to/apis/userHasAsset/USERID/ASSETID to get whether a user has an asset or not
