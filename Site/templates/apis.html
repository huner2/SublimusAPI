API List



User APIs


getUsernameById

Send a GET request to http://SublimusAPI.us.to/apis/getUsernameById/USERID to get username of a player by ID

getIdByUsername

Send a GET request to http://SublimusAPI.us.to/apis/getIdByUsername/USERNAME to get ID of a player by username

userCanManageAsset

Send a GET request to http://SublimusAPI.us.to/apis/userCanManageAsset/USERID/ASSETID to get whether a user can manage an asset or not

userHasAsset

Send a GET request to http://SublimusAPI.us.to/apis/userHasAsset/USERID/ASSETID to get whether a user has an asset or not

getFriendsOfUser

Send a GET request to http://SublimusAPI.us.to/apis/getFriendsOfUser/USERID/PAGENUMBER to get a list of the users friends on the current page, this will return an encoded JSON string that decodes as follows:

{
    {
        ["Id"] = long,
        ["Username"] = string,
        ["AvatarUri"] = string,
        ["AvatarFinal"] = boolean,
        ["IsOnline"] = boolean
    }
     -- And so on for remaining users
}

Asset APIs


getMarketplaceInfo

Send a GET request to http://SublimusAPI.us.to/apis/getMarketplaceInfo/ASSETID to get the info of an asset (marketplace, or not, this includes places), returned as a JSON encoded string that decodes as follows:

{
    ["AssetID"] = long,
    ["ProductID"] = long,
    ["Name"] = string,
    ["Description"] = string,
    ["AssetTypeId"] = integer,
    ["Creator"] = {
        ["Id"] = integer,
        ["Name"] = string
    }
    ["IconImageAssetId"] = long,
    ["Created"] = string,
    ["Updated"] = string,
    ["PriceInRobux"] = integer,
    ["PriceInTickets"] = null, -- Deprecated
    ["Sales"] = integer,
    ["IsNew"] = boolean,
    ["IsForSale"] = boolean,
    ["IsPublicDomain"] = boolean,
    ["IsLimited"] = boolean,
    ["IsLimitedUnique"] = boolean,
    ["Remaining"] = integer or null,
    ["MinimumMembershipLevel"] = integer,
    ["ContentRatingTypeId"] = integer
}

Clan APIs


getClanByUser

Send a GET request to http://SublimusAPI.us.to/apis/getClanByUser/USERID to get the properties of the user's clan returned as an encoded JSON string, that decodes as follows:

{
    ["Id"] = long,
    ["Name"] = string or null,
    ["EmblemAssetId"] = long
}

getClanById

Send a GET request to http://SublimusAPI.us.to/apis/getClanById to get the properties of the clan of the given id as an encoded JSON string, that decodes as follows:

{
    ["Id"] = long,
    ["Name"] = string or null,
    ["EmblemAssetId"] = long
}
